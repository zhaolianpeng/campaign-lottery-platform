# 月卡与战令（管理端只读）

## 1. 模块概述

| 项 | 说明 |
|----|------|
| 用户目标 | 查看当前战令赛季、任务与奖励配置（只读） |
| 入口 | `monthcard` Tab |
| API | `GET /api/v1/battle-pass/info`（复用 C 端接口） |

## 2. 界面清单

| 区块 | 内容 |
|------|------|
| 战令信息 | 赛季名、等级、任务列表、奖励表 |
| 月卡 | 展示性文案或状态（无编辑表单） |

## 3. 核心用户流程 **[部分实现]**

1. 进入 Tab → `battlePassQuery`
2. 只读浏览，无保存按钮
3. 配置变更需改种子数据或后端 **[规划中]** 专用 admin CRUD
